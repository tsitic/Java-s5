Код выводит:
static logic
logic (1) id= 0
logic (2) id= 1
ctor id= 100

Объяснение

static logic — статический блок выполняется один раз при первой загрузке класса
logic (1) id= 0 — первый инициализатор экземпляра, значение полю id присваивается по умолчанию (т.е. 0)
logic (2) id= 1 —  второй инициализатор запускается после того, как значение id было инициализировано в строке private int id = 1;
                   Поэтому к этому моменту id уже равно 1
ctor id= 100 — конструктор устанавливает значение id = 100 и выводит его в консоль
