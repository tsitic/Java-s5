1)
          interface Interface {
            ...
          }

Такой интерфейс доступен везде внутри того же пакета
Его нельзя использовать или реализовывать за пределами пакета

2)
          public interface Interface {
            ...
          }

Интерфейс с модификатором public имеет глобальную область видимости
Его можно использовать и реализовывать из любого другого класса или интерфейса, независимо от пакета

3)
          protected interface Interface {
            ...
          }

Такой модификатор недопустим для интерфейсов


4)
          private interface Interface {
            ...
          }

Такой модификатор недопустим для интерфейсов
